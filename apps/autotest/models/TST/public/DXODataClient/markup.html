Using UnityBase OData endpoint

This sample show usage of Developer Express grid mapped to UnityBase OData endpoint

UnityBase core module are used for authentication/authorization, so all network requests are secure.

Compatibility:
  • GET/POST/DELETE/PUT for CRUID operations - you can select/edit/add/remove grid rows
  • $filters (without any/all) - just set a filter on columns
  • $order - click columns caption for ordering
  • $select - only a subset of attributes are retrieved from entity
  • $top/$skip/$count - see a pagination
  • $metadata - click to
